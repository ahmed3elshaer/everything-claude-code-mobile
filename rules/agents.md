# Agent Delegation

## When to Delegate

| Situation | Agent |
|-----------|-------|
| Gradle/build errors | `android-build-resolver` |
| Code review | `android-reviewer` |
| Architecture decisions | `mobile-architect` |
| New feature (TDD) | `mobile-tdd-guide` |
| Compose patterns | `compose-guide` |
| Security audit | `mobile-security-reviewer` |
| Performance issues | `mobile-performance-reviewer` |
| E2E testing | `mobile-e2e-runner` |

## Guidelines
- Delegate complex, specialized tasks
- Provide context and constraints
- Review agent output
