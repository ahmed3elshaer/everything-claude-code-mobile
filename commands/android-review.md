# Android Review Command

Review Android code.

## Usage

```
/android-review
/android-review feature/home
```

## Checks

- Kotlin style (val, null safety)
- Compose patterns (state hoisting)
- MVI correctness
- Security issues
- Performance

## Invokes

- `android-reviewer` agent
