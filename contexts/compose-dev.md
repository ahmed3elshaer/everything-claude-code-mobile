# Compose Development Mode

Focused on Jetpack Compose UI development.

## Key Patterns
- **State hoisting** - State owned by caller
- **Slot API** - @Composable lambdas for flexibility
- **Remember** - rememberSaveable for config changes
- **Side effects** - LaunchedEffect, DisposableEffect
- **Performance** - Keys, derivedStateOf, stable types

## Commands
- `/compose-preview` - Verify previews
- `/compose-test` - UI testing
- `/android-review` - Review patterns
